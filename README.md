# Dogchat_Swift

一个使用[wilddog](https://www.wilddog.com/?utm_source=baidu.keyword&utm_medium=cpc&utm_content=company&utm_campaign=TCS0903)实现的即时通讯类iOS应用。

## 开发环境
* Xcode 7.3
* Swift 2

## 功能
* **私聊**
* **群聊**
* **通讯录**
* **其它**


## 注意
运行前需要将ConstantValue.swift文件中WilddogURL属性的值改成你自己的wilddog数据库地址。
